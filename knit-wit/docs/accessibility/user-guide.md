# Knit-Wit Accessibility Features - User Guide

## Overview

Knit-Wit is designed to be accessible to all crocheters, regardless of ability. This guide covers the accessibility features available in the app and how to use them.

## Quick Access

All accessibility features are available in **Settings > Appearance**:

1. Open the Knit-Wit app
2. Tap the **Settings** tab at the bottom
3. Scroll to the **Appearance** section

## Dyslexia-Friendly Font

### What is it?

A special font designed to make reading easier for people with dyslexia. The font (called OpenDyslexic) has weighted letter bottoms and unique letter shapes that help distinguish similar-looking letters.

### How to enable

1. Go to **Settings** > **Appearance**
2. Toggle **"Dyslexia-Friendly Font"** ON
3. All text in the app will immediately change to the easier-to-read font

**In Kid Mode:** This setting is called **"Easier Reading Font"** with simpler language.

### What it does

- Changes ALL text in the app to OpenDyslexic font
- Works on all screens: Home, Generate, Visualize, Export, Settings
- Applies to:
  - Pattern instructions
  - Button labels
  - Form fields
  - Settings descriptions
  - All other text

### Who benefits

- Users with dyslexia
- Users who struggle with letter recognition
- Users who find standard fonts hard to read
- Young learners still developing reading skills
- Anyone who prefers clearer letter shapes

### Tips

- Try it even if you don't have dyslexia - many users find it easier to read
- Combine with larger system font sizes (iOS/Android settings) for even better readability
- Works great in combination with Kid Mode for young learners

## Kid Mode

### What is it?

A simplified interface designed for young learners (ages 8-12) with larger buttons, friendly colors, and simpler language.

### How to enable

1. Go to **Settings** > **Appearance**
2. Toggle **"Kid Mode"** ON
3. The entire app transforms to be kid-friendly

### Features

- **Larger tap targets** (56×56 points minimum)
- **Simpler vocabulary**
  - "Make a Pattern" instead of "Generate Pattern"
  - "Ball" instead of "Sphere"
  - "Tube" instead of "Cylinder"
- **Friendly colors** (pink and yellow instead of purple)
- **Beginner-friendly instructions**
- **Animated tutorials** for complex concepts

### Combining with Dyslexia Font

Kid Mode and Dyslexia Font work great together:
1. Enable both features in Settings
2. Get larger buttons AND easier-to-read text
3. Perfect for young learners or beginners

## Dark Mode

### What is it?

A dark color scheme that's easier on the eyes in low-light conditions and may help with visual sensitivity.

### How to enable

1. Go to **Settings** > **Appearance**
2. Toggle **"Dark Mode"** ON
3. The app background becomes dark with light text

### Features

- Reduces eye strain in dark environments
- May help users with light sensitivity
- Works with Kid Mode (Dark Kid Mode)
- Works with Dyslexia Font

## Screen Reader Support

### Supported Screen Readers

- **iOS:** VoiceOver
- **Android:** TalkBack

### Features

- All buttons and controls have descriptive labels
- Settings changes are announced when toggled
- Forms have clear field labels and hints
- Images have alternative text descriptions
- Heading structure for easy navigation

### Tips for Screen Reader Users

1. **Navigating Screens:**
   - Swipe right to move to next item
   - Swipe left to move to previous item
   - Double-tap to activate buttons

2. **Using Forms:**
   - Each field is clearly labeled
   - Hints explain what to enter
   - Validation errors are announced

3. **Hearing Setting Changes:**
   - When you toggle settings, the app announces the change
   - Example: "Dyslexia-friendly font enabled"

## Color Contrast

### WCAG AA Compliance

All color combinations meet WCAG 2.1 Level AA standards:
- **Text contrast:** 4.5:1 minimum
- **UI element contrast:** 3:1 minimum

### High Contrast Mode

The app respects system-level high contrast settings on iOS and Android.

## Font Size Scaling

### System Font Size

The app respects your device's system font size setting:

**iOS:**
1. Go to **Settings** > **Display & Brightness** > **Text Size**
2. Adjust slider to preferred size
3. Return to Knit-Wit - all text will scale

**Android:**
1. Go to **Settings** > **Display** > **Font Size**
2. Select preferred size
3. Return to Knit-Wit - all text will scale

### Recommended Sizes

- **Standard:** Default system size
- **Large:** +1 or +2 steps for easier reading
- **Extra Large:** +3 steps for low vision users

## Touch Targets

### Minimum Sizes

All interactive elements meet accessibility guidelines:

- **Standard Mode:** 44×44 points (Apple HIG)
- **Kid Mode:** 56×56 points (extra large for small hands)

### Examples

- All buttons are easy to tap
- Form fields have large touch areas
- Switches and toggles are appropriately sized

## Troubleshooting

### Font not changing

**Problem:** Toggled dyslexia font but text looks the same

**Solution:**
1. Check that toggle is ON (switch should be colored)
2. Try navigating to a different screen and back
3. Close and reopen the app
4. If still not working, report as a bug

### Text too small/large

**Problem:** Font size is uncomfortable to read

**Solution:**
1. Use your device's system font size settings
2. Try combining with dyslexia font for clearer letters
3. Use pinch-to-zoom on pattern visualizations

### Screen reader not announcing

**Problem:** Screen reader not reading content

**Solution:**
1. Ensure VoiceOver (iOS) or TalkBack (Android) is enabled in device settings
2. Check that Knit-Wit has accessibility permissions
3. Try toggling screen reader off and on
4. Update to latest app version

### Colors too bright/dark

**Problem:** Colors are hard to see

**Solution:**
1. Try toggling Dark Mode
2. Use device high contrast mode
3. Adjust device brightness
4. Provide feedback for future colorblind modes

## Providing Feedback

We're committed to making Knit-Wit accessible to everyone. If you encounter accessibility barriers or have suggestions:

1. **Report Issues:** Use Settings > About > Report an Issue
2. **Describe the barrier:** What feature isn't working for you?
3. **Include device info:** iOS/Android version, screen reader used, etc.
4. **Suggest improvements:** What would help you use the app better?
