# Font Assets

This directory contains custom fonts for the Knit-Wit mobile app.

## OpenDyslexic Font

The OpenDyslexic font is used to improve readability for users with dyslexia.

### Installation Instructions

1. Download OpenDyslexic fonts from: https://opendyslexic.org/
2. Place the following files in this directory:
   - `OpenDyslexic-Regular.ttf`
   - `OpenDyslexic-Bold.ttf`

### License

### Files Required

```
assets/fonts/
├── OpenDyslexic-Regular.ttf  (Required - download from opendyslexic.org)
├── OpenDyslexic-Bold.ttf     (Required - download from opendyslexic.org)
└── README.md                 (This file)
```

### Usage

The font is automatically loaded by Expo and can be toggled in Settings > Appearance > Dyslexia-Friendly Font.
